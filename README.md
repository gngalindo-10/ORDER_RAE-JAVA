## 📋 Descripción del Proyecto

El sistema **ORDER_RAE** se consolida como una solución tecnológica integral orientada a la optimización y automatización de los procesos operativos y administrativos de LA SUPER BODEGA DEL MUEBLE. Su implementación permite centralizar la gestión de inventarios, ventas, clientes y producción, garantizando la integridad, disponibilidad y trazabilidad de la información en tiempo real.

### 🎯 Objetivos
- Automatizar el registro y seguimiento de pedidos
- Centralizar la información de clientes y productos
- Generar reportes para la toma de decisiones
- Implementar arquitectura MVC con Spring Boot
- Garantizar integridad y consistencia de datos
